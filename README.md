# 心理学科普（psychology-site）

这是一个用 VitePress 构建的静态内容网站，用于普及心理学与认知科学中的基础概念与常见发现。内容以结构化的阅读笔记与专题指南为主，帮助读者更清晰地理解认知、行为、情绪、关系与心理健康相关主题，并尽量做到基于证据、可复述、可应用。

## 内容概览

- 基础入门：从日常现象出发理解核心概念与机制
- 心智与大脑：知觉、记忆、情绪与神经系统等
- 情绪与关系：情绪识别与调节、沟通与关系模式
- 社会心理与应用：群体与规范、偏差，以及实践场景

站点内容位于 `docs/`，首页为 `docs/index.md`。

## 本地开发

环境：需要 Node.js（建议使用较新的 LTS 版本）与 npm。

```bash
npm install
npm run docs:dev
```

启动后按终端提示打开本地地址进行预览。

## 构建与预览

```bash
npm run docs:build
npm run docs:preview
```

## 项目结构

- `docs/`：站点内容与导航
- `docs/guide/`：专题与指南页面
- `docs/.vitepress/`：VitePress 配置与主题扩展

## 写作约定（建议）

- 以“可验证、可复述、可应用”为目标组织内容
- 尽量给出概念边界与适用条件，避免把个体经验当作普遍结论
- 单篇保持短小清晰，优先提供框架与关键结论
