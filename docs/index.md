## 基础入门到底学什么

这里的“基础”不是研究流程，而是帮你理解日常现象的底层逻辑。

- 为什么记忆会失真、我们会高估自己
- 为什么习惯难改、奖励有时会失效
- 为什么情绪被触发、压力难以消退
- 为什么同一句话在不同关系里效果不同
