# 发展与人格

发展心理学关心心智如何成长，人格心理学关心个体差异如何稳定与变化。

## 发展轨迹

- 早期依恋影响安全感与信任
- 青少年期关注身份探索与自我定位
- 成年期强调关系、意义与长期目标

## 人格结构

- 大五人格是常用的描述框架
- 人格倾向与环境相互作用

## 风险与韧性

- 风险来自压力源与资源不足
- 韧性源于支持系统与自我调节

## 原生家庭的影响

- 依恋模式：影响信任、亲密与安全感
- 沟通方式：塑造表达习惯与冲突处理
- 边界与规则：影响自我控制与责任感
- 角色期待：影响自我价值与成就动机
- 情绪氛围：影响情绪识别与调节能力

## 实践提示

- 用“阶段性任务”理解成长挑战
- 关注优势而非只看问题
